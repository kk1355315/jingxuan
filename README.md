## 📱 导出PDF说明

### ⚠️ 重要提示
为确保最佳的PDF导出效果，请严格按照以下设置操作：

### 推荐设置（PPT横向格式）
- **页面大小**：A4
- **方向**：**横向**（重要！）
- **边距**：10mm（或选择"默认"）
- **缩放**：100%（不要选择"适合页面"）
- **背景图形**：**必须启用**（重要！）
- **页眉页脚**：关闭

### 详细导出步骤

#### Chrome浏览器（推荐）
1. 点击页面右上角的"导出PDF"按钮
2. 在打印对话框中：
   - 目标：选择"另存为PDF"
   - 页面：选择"全部"
   - 布局：**横向**（重要！）
   - 颜色：彩色
   - **更多设置**：
     - 纸张尺寸：A4
     - 边距：默认
     - 选项：**勾选"背景图形"**
3. 点击"保存"

#### Edge浏览器
1. 按 `Ctrl+P` 或点击导出按钮
2. 打印机：选择"Microsoft Print to PDF"
3. 设置：
   - 方向：**横向**（重要！）
   - 颜色：彩色
   - **更多设置**中勾选"背景图形"
4. 点击"打印"

### 预期效果
正确设置后，PDF将包含：
- ✅ 8页内容，每页一个主要区块
- ✅ 完整的渐变背景色
- ✅ 彩色图标和按钮
- ✅ 所有视觉效果和样式
